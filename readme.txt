Таймеры:
1. вывод звука
	-делитель 80000
	TIM16 
2. чтение нот и воспроизведение

3. запуск ADC и сохранение температуры в переменных
+ 2. проверка клавиатуры и индикация светодиодов
	
3. управление ЖКИ
4. мигание светодиодов, индикации

Задачи:
- основная (выбор режимов, настройка параметров)
- вывод индикации часов	
- для каждой программы (контроль температуры, времени, вывод индикации)

Функции:
+LedControl(led,on/off/blink)
	включает выключает или контролирует мигание светодиодов
-LCDModeOut(mode, on/off/blink)
	включает выключает мигает индикацией режимов или цифровими индикаторами
+LCD_WriteTime(hour,min)	
	установка индикации минут, часов
-LCDOn
	включение индикации ЖКИ
-LCDOff
	выключение индикации ЖКИ
-TEHBottomState  состояние нижнего ТЭНа
-TEHMidState  состояние среднего ТЭНа
-TEHTopState  состояние верхнего ТЭНа
-ErrorHandle
	мониторинг ошибок датчиков и пр.

-PowerSleep()
	переход в спящий режим
	все выводы в аналоговый вход (экономия)
	выключение всей индикации, кроме часов
	тактовая частота 512 кГц
	засыпание 
	просыпание по таймеру для вывода времени 100раз в сек
-PowerOn()
	переход в рабочий режим
	частота 24 MГц
	включение индикации, кнопок, звука
	
+ Beep 
	вывод звука сопровождения нажатия клавиши
- SoundPlay
	проиграть выбранную мелодию  (отключается любым нажатием клавиши)
-TempConvert
	пересчет значения ADC в значение температуры
-TempGetTop
	значение температуры верхнего датчика
-TempGetBottom
	значение температуры нижнего датчика
	
Макросы
-TEHBottomOn  включить нижний ТЭН
-TEHBottomOff  выключить нижний ТЭН
-TEHMidOn  включить средний ТЭН
-TEHMidOff  выключить средний ТЭН
-TEHTopOn  включить верхний ТЭН
-TEHTopOff  выключить верхний ТЭН
